+ 版本退回
    * git reset
    * git reset --hard HEAD^ 上一个版本
    * git reset --hard HEAD^^ 上上一个版本
    * git reset --hard HEAD~100 上100个版本
    * git reset --hard 29d7q 回到版本号为的29d7q版本
+ git log 查看提交信息 用于回退到以前版本的版本号查看
+ git reflog 查看所有提交信息 用于返回到最新版本的版本号查看
+ Git tracks changes.
